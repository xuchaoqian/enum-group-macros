# enum-group-macros

Define grouped enums with ergonomic pattern matching in Rust.

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
enum-group-macros = { git = "https://github.com/xuchaoqian/enum-group-macros.git" }
```
